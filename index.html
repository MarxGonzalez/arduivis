        Compatibility

        This is a list of motors, sensors, breakout board, etc. which have been used with arduivis

      • Senors
        • MPU-6050 (3-axis gyroscope/3-axis accelerometer)
        • Temperature Sensor (DS18B20) (Waterproof)
        • Flex Sensor (various companies)
        • Force Sensitive Resistor (various companies)
        • Mini Photocell (various companies)
        • Electret Microphone Breakout
        • PIR Motion Sensor
        • Ultrasonic Range Finder(LV-MaxSonar-EZ0)
        • Ambient Light Sensor(TEMPT600)
        • SparkFun Altitude/Pressure Sensor Breakout (MPL3115A2)
        • SparkFun Barometric Pressure Sensor Breakout (BMP180)
        • SparkFun Humidity Sensor Breakout (HIH-4030)
        • Ultrasonic Distance Sensor (HC-SR04)
        • Linear SoftPot (Ribbon Sensor)
        • Circular SoftPot (Ribbon Sensor)
      • Wireless
        • SparkFun XBee Explorer USB
        • XBee 2mW Wire Antenna - Series 2 (ZigBee Mesh) (RF)
        • HC-05 (Bluetooh)
        • Bluefruit EZ-Link (Bluetooh)
        • WiFi Breakout w/Ceramic Antenna(CC3000)
      • Breakout Boards
        • MP3 Trigger
        • Sparkfun MP3 Player Shield
        • SparkFun LED Driver Breakout (TLC5940)
        • SparkFun Weather Band Receiver Breakout (Si4707)
        • Breakout Board for Thumb Joystick
        • Stereo 3.7W Class D Audio Amplifier (MAX98306)
      • Components
        • 16-Bit ADC (4 Channel w/ Programmable Gain Amplifier) (ADS1115)
        • 12-Bit ADC (4 Channel w/ Programmable Gain Amplifier) (ADS1015)
        • 8-Channel 10-Bit ADC w/SPI Interface (MCP3008)
        • 12-Bit DAC w/I2C Interface (MCP4725)
        • 4-Channel 12-Bit DAC w/I2C Interface (MCP4728)
        • 8-Channel 12-Bit DAC w/SPI Interface (MCP3208)
        • Line Laser Diode - (5mW 650nm Red)
      • LEDs/LCDs
        • Digital RGB LED Weatherproof Strip (LPD8806)
        • Adafruit Mini 8x8 LED Matrix (I2C Backpack)
        • Adafruit Bicolor LED Square Pixel Matrix (I2C Backpack)
        • Graphic LCD 84x48 (Nokia 5110)
        • Basic 16x2 Character LCD
      • Motors
        • Servo (various companies)
        • Stepper (various companies)
        • Adafruit 16-Channel 12-bit PWM/Servo Driver (PCA9685 - I2C interface)
        • EasyDriver - Stepper Motor Driver
