Arduino

  • Find the arduivis root folder and select the MaxMSP folder
  • Open the folder named "model1_output"
  • Within this folder, find the folder named "arduivis_MaxMSP_model1_output"
  • Open the Arduino sketch and upload it to the Arduino board

MaxMSP

  • Find the arduivis root folder and select the MaxMSP folder
  • Open the folder named "model1_output"
  • Within this folder, open the dedicated MaxMSP patch named arduivis_MaxMSP_model1_output.maxpat
